Model.py Documentation¶

Access the model.py folder in Tensorflow Implementation folder. The model.py file has 2 important functions that are pertinent to creation of the model.

model.create_model(num_tags)¶

This function calculates the masked loss to reduce the effect of loss due to padding on the model.

Parameters

num_tags – the total number of unique tags, defaults to none

Returns

model: returns the BERT model

Return type

tensorflow model

model.masked_ce_loss(real, pred)¶

This function calculates the masked loss to reduce the effect of loss due to padding on the model.

Parameters

  • real – Actual Label in the processed data, defaults to none

  • pred – Predicited Label, defaults to none

Returns

loss_: returns the mean loss

Return type

float
